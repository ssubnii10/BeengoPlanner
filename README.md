# BeengoPlanner 🗓️

취준생을 위한 스마트 일정 관리 앱입니다.
AI 일정 파싱, 리포트, 테마 커스터마이징 등 취준 생활에 필요한 기능을 직접 기획하고 개발했습니다.

## 주요 기능

- 📅 일정 관리 — 취업 준비 일정을 한눈에 관리
- 🤖 AI 파싱 — 공고 텍스트를 붙여넣으면 일정 자동 생성
- 📊 리포트 — 취준 활동 통계 및 진행 현황 확인
- 🎨 테마 커스터마이징 — 다양한 테마로 나만의 캘린더

## 기술 스택

- React Native (Expo)
- TypeScript
- Expo Router

## 시작하기

### 1. 의존성 설치
```bash
npm install
```

### 2. 앱 실행
```bash
npx expo start
```

실행 후 아래 방법으로 앱을 열 수 있어요.

- **실제 기기** — Expo Go 앱 설치 후 QR 코드 스캔
- **Android 에뮬레이터** — Android Studio 필요
- **iOS 시뮬레이터** — Mac + Xcode 필요
